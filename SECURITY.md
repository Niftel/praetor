# Security Policy

## Supported versions

Praetor is under active development. Security fixes are applied to the current
`main` branch and to the latest development release candidate.

## Reporting a vulnerability

Please report suspected vulnerabilities through
[GitHub private vulnerability reporting](https://github.com/Niftel/praetor/security/advisories/new).
Do not open a public issue for an undisclosed security vulnerability.

Include the affected component and version, reproduction steps, expected impact,
and any suggested mitigation. Reports will be acknowledged as soon as practical,
and coordinated disclosure will be agreed with the reporter before publication.
